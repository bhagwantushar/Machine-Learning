# Machine Learning Coursework

## About this
This repository contains coursework from my Machine Learning module.

The assignment covers different concepts including probabilistic modelling, regression, classification, and deep learning.

---

## Part 1 – Tracking Bees

In this part, I worked on reconstructing the flight path of a bee using limited observations.

Key concepts:
- likelihood and negative log likelihood
- estimating position from directional data
- Gaussian noise modelling

I implemented functions to:
- calculate likelihood
- predict positions
- optimise parameters
- compare predicted vs actual path

---

## Part 2 – Regression

Used Gaussian basis functions to predict the bee’s trajectory over time.

Learned:
- how regression models work
- how basis functions affect predictions
- importance of parameter tuning

---

## Part 3 – Classification

Implemented different models on image data:

- Logistic Regression
- Random Forest
- Convolutional Neural Network (CNN)

Compared:
- training vs test accuracy
- performance of different models

---

## Part 4 – Denoising Autoencoder

Worked with CIFAR-10 dataset to:
- add noise to images
- train a neural network to remove noise

This helped me understand:
- feature learning
- neural network training
- reconstruction error

## What I learned

- how probabilistic models work in practice
- difference between regression and classification
- how deep learning models are trained
